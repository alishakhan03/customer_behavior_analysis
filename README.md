# Data Analytics Project: Customer Behavior Analysis

## Overview

This project demonstrates an end-to-end Data Analytics workflow, starting from data collection and preprocessing to SQL analysis, dashboard development, and business reporting. The objective was to extract meaningful insights from customer purchasing behavior and present them through interactive visualizations and actionable recommendations.

The project covers:

* Data loading and preprocessing using Python
* Exploratory Data Analysis (EDA)
* Data cleaning and transformation
* SQL-based data analysis using PostgreSQL/MySQL/SQL Server
* Interactive dashboard creation in Power BI
* Business report generation
* Presentation design using Gamma AI



## Dataset

The dataset contains customer shopping and purchasing information, including demographics, product categories, purchase history, payment methods, subscription status, ratings, and shopping preferences.

3. Execute SQL scripts available in the project.
